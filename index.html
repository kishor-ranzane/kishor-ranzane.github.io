CXone FDL - Federated Data Layer

NICE Systems Team Size: 11 Customer Experience

FDL is an AI-powered core data platform within the NICE CXone Illuminate domain designed to provide a unified, secure, and scalable access layer over multiple heterogeneous data sources. The platform combines traditional data federation with LLM-based prompt management for intelligent data querying and analytics. The system exposes standardized REST APIs that abstract underlying data sources including Snowflake, Google Analytics, Salesforce, and Bot Builder, allowing consumers to query operational and analytical data without direct dependency on physical storage.

Key Features

  • Federated Data Access Layer: Unified REST API interface over multiple backend data sources (Snowflake, Google Analytics, Salesforce, Bot Builder) without data duplication
  • LLM Integration: Integrated Prompt Manager Service for AI-driven operations with configurable prompt templates, versioning, and token usage tracking
  • Tenant & Division-Level Segmentation: Strict data isolation through automatic tenant_id injection using Apache Calcite SQL transformation with RBAC support
  • High-Performance Query Pipeline: SQL processing with syntax validation, semantic validation, guardrailing, and optimized Snowflake execution
  • AI-Ready Schema Architecture: Schema definitions with instructions and examples for LLM context and few-shot learning
  • Bugfixer Agent: Autonomous AI agent for code analysis, bug detection, and automated fix generation using LLM-powered reasoning

Key Responsibilities

  • Designed and developed REST APIs for fetching data from Snowflake, Google Analytics, Salesforce, and Bot Builder
  • Implemented LLM integration layer with retry-resilient REST client, exponential backoff, and distributed tracing
  • Built tenant and division-based data segmentation using Apache Calcite SQL transformation
  • Developed Bugfixer Agent using Agentic AI principles for autonomous code analysis and bug resolution
  • Integrated AWS services (S3, Secrets Manager, Systems Manager, DynamoDB, CloudWatch)
  • Implemented comprehensive unit tests, integration tests (Cucumber BDD), and mutation testing (PIT)

WCAO - World Class Account Onboarding

HSBC via Synechron Team Size: 17 Banking

WCAO is a microservices-based digital onboarding platform for HSBC that enables customers to open bank accounts within five minutes. The system integrates multiple identity verification services, including government (UIDAI) and third-party providers (SIGNZY), and uses an event-driven AWS architecture with orchestrated workflows to ensure secure, scalable, and compliant customer onboarding across multiple countries.

Key Features

  • 5-Minute Account Opening: Streamlined digital onboarding reducing account creation time drastically
  • Multi-Provider Identity Verification: Integrated UIDAI (government) and SIGNZY for comprehensive KYC
  • Event-Driven Architecture: AWS-based event-driven system with orchestrated workflows
  • Multi-Country Support: Scalable platform supporting onboarding across multiple regions
  • Compliance & Security: Built-in regulatory compliance and secure data handling
